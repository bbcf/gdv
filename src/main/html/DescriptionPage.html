Description

GDV is a fast and easy to use genome browser. The main goal is to provide a tool for biologist and bioanalysts who want to VISUALIZE and ANALYSE their data in an interactive and responding way.
In order do such job, GDV will preprocess each inputs and build a view with. Then the view provided is accessible at any time.

Technologies

  • written in JAVA .
  • built with Apache wicket.
  • served by Apache Tomcat.
  • viewer in Javascript from JBrowse engine.
  • database in postgresql.

    • Features provided

  • based on the last web-features HTML5.
  • user management (group in a near future).
  • secure login (see TEQUILA.
  • tracks management (saving files after user navigation).
  • different input formats accepted (GFF, WIG, Bedgraph,BED) in different ways (form URL or file upload).
  • Planning to add BAM & SAM support.
  • creation of projects and tracks from programmatic access (pipelines) - restricted feature, ask for collaboration-

    • Architecture

    GDV is composed by 3 modules:

  • JBrowsoR, built on top of JBrowse, is a job manager that interacts with JBrowse.
  • GeneRep stores information about the genomes and assemblies for GDV.
  • GFeatMiner is a statistical and data mining module that will be integrated to GDV.
    • GDV has to communicate with those modules and provide a user friendly interface.
